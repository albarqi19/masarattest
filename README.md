# موقع مسارات عدلية للمحاماة

إعادة بناء كاملة لموقع [masaradl.sa](https://www.masaradl.sa) ككود مصدري ثابت (HTML/CSS/JS) بدون أي ارتباط بمنصة Wix — جاهز للاستضافة على Vercel أو أي استضافة أخرى.

## هيكل المشروع

```
├── index.html        الرئيسية
├── about-us.html     من نحن
├── services.html     خدماتنا
├── team.html         المؤسسون (فريق العمل)
├── blog.html         المدونة
├── faq.html          الأسئلة الشائعة
├── contact.html      تواصل معنا
├── terms.html        الشروط والأحكام
├── privacy.html      سياسة الخصوصية
├── vercel.json       إعدادات Vercel (روابط نظيفة + تحويلات)
└── assets/
    ├── css/style.css خصائص التصميم (الألوان والهوية)
    ├── js/main.js    القوائم والعدادات والحركات
    ├── fonts/        خط عام الإبل (Bold / ExtraBold)
    └── img/          الشعار والصور وشعارات الاعتمادات
```

## المعاينة محلياً

افتح `index.html` مباشرة في المتصفح، أو شغّل خادماً محلياً:

```bash
python -m http.server 8000
# ثم افتح http://localhost:8000
```

## النشر على Vercel

1. أنشئ حساباً على [vercel.com](https://vercel.com)
2. ثبّت أداة Vercel: `npm i -g vercel`
3. من داخل مجلد المشروع نفّذ: `vercel --prod`
4. اربط النطاق `masaradl.sa` من لوحة تحكم Vercel: **Settings → Domains**
5. حدّث سجلات DNS للنطاق (A record إلى `76.76.21.21` أو CNAME إلى `cname.vercel-dns.com`)

> بديل: ارفع المجلد إلى مستودع GitHub ثم استورده في Vercel — كل تحديث للمستودع يُنشر تلقائياً.
